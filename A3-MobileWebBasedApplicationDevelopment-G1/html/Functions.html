Introduction

Functions in JavaScript are blocks of codes that are designed to perform particular tasks that can be resued mulitple time. Parameters are passed through function signature without specifying the return type. JavaScript is flexible and allows user to write function without a return value.

Advantages

Advantages of Functions in Javascript are:
  • Functions in Javascript are used to encourgae the reusability of code which can simplify the debugging

Disadvantages

  • If functions are implemented improperly than the code can be hard to maintain and can cause potential memory leaks

Usages

  • Functions are used in JavaScript to perform calculations, manipulating the data and in handling events
