Introduction

The Document Object Model which is also know as DOM is a programming interface especially for HTML, Javascript, XML and so on documents. It basically represents how the structure of a document in tree format which consists of objects that allows prgrams to dynamically access and update the content and style of the page. The DOM acts as a way for JavaScript to interact with the webpage

Advantages

Advantages of DOM are
  • It enables content to dynamically update without page being able to reload
  • DOM also helps in enhancing the interactivity with the webpage by allowing the real time changes to the structure of the document which provides a standard way in order of access and manipulating the web pages accross multiple browsers

Disadvantages

  • Manipulating the DOM can cause performance issues in large documents, frequent updates can cause the pages to slow down
  • More complex the DOM is more harder it will be to maintain the code

Usages

  • DOM is used to create, modify and remove the elements and it also handles events like on click in HTML and it is also used to manipulate the CSS and Javascript
  • DOM is fundamental while building an interactive web applications
