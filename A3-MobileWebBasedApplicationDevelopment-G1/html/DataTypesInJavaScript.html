Introduction

JavaScript supports multiple data types which can be divided into primitive data types and objects. Primitive data types basically means string, number, boolean, null, undefined and symboles. Object data types includes arrays, functions and regular expressions

Advantages

Advantages of Data Types in Javascript are:
  • JavaScript allows dynamically to variables to hold different types of data at different times which offers flexibility
  • Due to varity in datatypes diverse operations are easily supported which leads to simple arithmetic operations to complex data operations

Disadvantages

  • Due to dynamic typing unexpected behaviours can cause data coercion where JavaScript can converts from one type to another automatically. This can be pain to trace bugs
  • Due to lack of strong typing large codebases are harder to maintain

Usages

  • Datatypes are used in variable declarations, function parameters and to return values.
  • Datatypes are fundamental in controlling the flow of a program
